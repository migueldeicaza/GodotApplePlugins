GKLeaderboardSet
================

**Inherits:** :ref:`RefCounted<class_RefCounted>` **<** :ref:`Object<class_Object>`

Represents a Game Center leaderboard set.

.. rst-class:: classref-introduction-group

Description
-----------

``GKLeaderboardSet`` groups related leaderboards in Game Center. Use :ref:`load_leaderboard_sets()<class_GKLeaderboardSet_method_load_leaderboard_sets>` to fetch available sets, then :ref:`load_leaderboards()<class_GKLeaderboardSet_method_load_leaderboards>` to enumerate the boards inside a set.
