GKVoiceChat
===========

**Inherits:** :ref:`RefCounted<class_RefCounted>` **<** :ref:`Object<class_Object>`

Represents a Game Center voice chat channel.

.. rst-class:: classref-introduction-group

Description
-----------

Use :ref:`GKMatch.voice_chat()<class_GKMatch_method_voice_chat>` to open a voice channel for a realtime match, then call :ref:`start()<class_GKVoiceChat_method_start>` / :ref:`stop()<class_GKVoiceChat_method_stop>` and monitor :ref:`player_state_changed<class_GKVoiceChat_signal_player_state_changed>`.
