ARHandSkeleton
==============

**Inherits:** :ref:`RefCounted<class_RefCounted>` **<** :ref:`Object<class_Object>`

Provides joint transforms for a tracked hand skeleton.

.. rst-class:: classref-introduction-group

Description
-----------

Contains 27 joint transforms (wrist through forearm) for a tracked hand. Access individual joints with :ref:`get_joint_transform()<class_ARHandSkeleton_method_get_joint_transform>` or get all at once with :ref:`get_all_joint_transforms()<class_ARHandSkeleton_method_get_all_joint_transforms>`. **visionOS only.** On iOS and macOS, this type exists for API consistency but returns empty/default values.
