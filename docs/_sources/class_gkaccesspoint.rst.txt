GKAccessPoint
=============

**Inherits:** :ref:`RefCounted<class_RefCounted>` **<** :ref:`Object<class_Object>`

Controls the Game Center access point overlay.

.. rst-class:: classref-introduction-group

Description
-----------

The access point displays a control in a corner of your game that opens a Game Center dashboard when the player taps or clicks it.

Use the :ref:`GameCenterManager.access_point<class_GameCenterManager_property_access_point>` property to get the shared access point object, then configure its :ref:`location<class_GKAccessPoint_property_location>`, :ref:`active<class_GKAccessPoint_property_active>`, and other properties.
