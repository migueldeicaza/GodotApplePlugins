ARWorldMap
==========

**Inherits:** :ref:`RefCounted<class_RefCounted>` **<** :ref:`Object<class_Object>`

A serializable snapshot of the AR world state for session persistence.

.. rst-class:: classref-introduction-group

Description
-----------

Captures the spatial mapping, anchors, and feature points of an AR session so it can be saved and restored later. Use :ref:`ARSession.get_current_world_map()<class_ARSession_method_get_current_world_map>` to capture and :ref:`ARSession.run_with_world_map()<class_ARSession_method_run_with_world_map>` to restore. **iOS only.** On visionOS, spatial anchors are persisted automatically by the system via WorldTrackingProvider. On macOS, world map persistence is not available through remote sessions. The type exists on all platforms for API consistency but serialize/deserialize are no-ops outside iOS.
