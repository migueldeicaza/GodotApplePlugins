ARGeoAnchor
===========

**Inherits:** :ref:`RefCounted<class_RefCounted>` **<** :ref:`Object<class_Object>`

A GPS-positioned anchor for geo-referenced AR content.

.. rst-class:: classref-introduction-group

Description
-----------

Represents a world position defined by latitude, longitude, and altitude. Used with :ref:`ARSession.run_geo_tracking()<class_ARSession_method_run_geo_tracking>`. **iOS 14+ only.** On visionOS and macOS, this type exists for API consistency but is never populated with real tracking data.
