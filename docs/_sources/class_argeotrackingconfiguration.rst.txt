ARGeoTrackingConfiguration
==========================

**Inherits:** :ref:`RefCounted<class_RefCounted>` **<** :ref:`Object<class_Object>`

Configuration for GPS-anchored geo tracking AR sessions.

.. rst-class:: classref-introduction-group

Description
-----------

Configures a session for geo tracking, which combines visual localization with GPS to place content at real-world geographic coordinates. Use with :ref:`ARSession.run_geo_tracking()<class_ARSession_method_run_geo_tracking>`. Call :ref:`check_availability()<class_ARGeoTrackingConfiguration_method_check_availability>` to verify the current location supports geo tracking before starting. **iOS 14+ only.** On visionOS and macOS, :ref:`is_supported()<class_ARGeoTrackingConfiguration_method_is_supported>` returns false.
