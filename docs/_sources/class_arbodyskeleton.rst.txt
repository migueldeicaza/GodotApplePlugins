ARBodySkeleton
==============

**Inherits:** :ref:`RefCounted<class_RefCounted>` **<** :ref:`Object<class_Object>`

Provides joint transforms for a tracked body skeleton.

.. rst-class:: classref-introduction-group

Description
-----------

Contains the 3D joint transforms for a tracked body. Access joints by index using :ref:`get_joint_transform()<class_ARBodySkeleton_method_get_joint_transform>` and :ref:`get_joint_name()<class_ARBodySkeleton_method_get_joint_name>`. **iOS only.** On visionOS and macOS, this type exists for API consistency but returns empty/default values.
