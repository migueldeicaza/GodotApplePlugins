GKMatchmakerViewController
==========================

**Inherits:** :ref:`RefCounted<class_RefCounted>` **<** :ref:`Object<class_Object>`

Presents Apple's Game Center matchmaking UI.

.. rst-class:: classref-introduction-group

Description
-----------

Use this helper to show the stock ``GKMatchmakerViewController`` workflow described in ``GameCenterGuide.md``. You can either create and present the controller manually for full control or call :ref:`request_match()<class_GKMatchmakerViewController_method_request_match>` for the convenience flow demonstrated in the guide. See `Apple's GKMatchmakerViewController reference <https://developer.apple.com/documentation/gamekit/gkmatchmakerviewcontroller>`__ for Apple's API documentation.



Convenience usage from the guide:


.. tabs::

 .. code-tab:: gdscript

    var request := GKMatchRequest.new()
    request.max_players = 2
    request.min_players = 1
    request.invite_message = "Join me in a quest to fun"

    GKMatchmakerViewController.request_match(request, func(game_match: GKMatch, error: Variant) -> void:
        if error:
            print("Could not request a match %s" % error)
        else:
            print("Match ready")
            # Store game_match and connect to its signals as needed.
    )
