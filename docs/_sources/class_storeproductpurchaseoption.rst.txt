StoreProductPurchaseOption
==========================

**Inherits:** :ref:`RefCounted<class_RefCounted>` **<** :ref:`Object<class_Object>`

Represents an option that can be supplied when purchasing a :ref:`StoreProduct<class_StoreProduct>`.

.. rst-class:: classref-introduction-group

Description
-----------

You create instances of this class and you can pass those to :ref:`StoreKitManager.purchase_with_options()<class_StoreKitManager_method_purchase_with_options>`.
