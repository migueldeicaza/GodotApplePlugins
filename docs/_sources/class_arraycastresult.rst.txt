ARRaycastResult
===============

**Inherits:** :ref:`RefCounted<class_RefCounted>` **<** :ref:`Object<class_Object>`

Contains the result of an AR raycast, including the hit transform and target surface.

.. rst-class:: classref-introduction-group

Description
-----------

Returned by :ref:`ARSession.raycast()<class_ARSession_method_raycast>`. Contains the world transform of the hit point, the type of surface hit, and an optional anchor reference. **iOS only.** On visionOS/macOS this type exists for API consistency but is never populated with real data.
