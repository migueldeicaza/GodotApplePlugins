ARBodyTrackingConfiguration
===========================

**Inherits:** :ref:`RefCounted<class_RefCounted>` **<** :ref:`Object<class_Object>`

Configuration for body tracking AR sessions.

.. rst-class:: classref-introduction-group

Description
-----------

Configures a session for full-body skeletal tracking. Use with :ref:`ARSession.run_body_tracking()<class_ARSession_method_run_body_tracking>`. **iOS only.** On visionOS and macOS, :ref:`is_supported()<class_ARBodyTrackingConfiguration_method_is_supported>` returns false and the configuration is inert.
