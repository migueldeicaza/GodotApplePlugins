ARBodyAnchor
============

**Inherits:** :ref:`RefCounted<class_RefCounted>` **<** :ref:`Object<class_Object>`

Represents a tracked human body with skeletal joint data.

.. rst-class:: classref-introduction-group

Description
-----------

Provides the root pose, tracking state, estimated scale factor, and optional :ref:`ARBodySkeleton<class_ARBodySkeleton>` for a detected body. Delivered through anchor signals when using :ref:`ARSession.run_body_tracking()<class_ARSession_method_run_body_tracking>`. **iOS only.** On visionOS and macOS, this type exists for API consistency but is never populated with real data.
