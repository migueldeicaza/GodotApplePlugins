GKChallenge
===========

**Inherits:** :ref:`RefCounted<class_RefCounted>` **<** :ref:`Object<class_Object>`

**Inherited By:** :ref:`GKAchievementChallenge<class_GKAchievementChallenge>`, :ref:`GKScoreChallenge<class_GKScoreChallenge>`

Represents a Game Center challenge sent between players.

.. rst-class:: classref-introduction-group

Description
-----------

Base wrapper for challenge objects delivered by Game Center. Use :ref:`load_received_challenges()<class_GKChallenge_method_load_received_challenges>` to fetch outstanding challenges for the local player, inspect :ref:`state<class_GKChallenge_property_state>` and :ref:`challenge_type<class_GKChallenge_property_challenge_type>`, and cast to :ref:`GKScoreChallenge<class_GKScoreChallenge>` or :ref:`GKAchievementChallenge<class_GKAchievementChallenge>` when you need type-specific data.
